# CurChanger
Программа которая способна переводить валюты 


Чтобы запустить программу нужно 
1. Прописать в консоли pip install -r requirements.txt
2. Вписать ваш api ключ с сайта exchangerate-api.com в файле server.py в поле YOUR_API_KEY
3. Запустить две консоли в папке с файлами
4. Прописать в первой консоли python server.py
5. Прописать во второй консоли python client.py
6. Наслаждаться сделанным на коленки проектом
